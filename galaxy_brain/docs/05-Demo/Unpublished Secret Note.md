# Unpublished Secret Note

This note does NOT have `publish: true` in frontmatter.

It will appear as a **ghost node** (white sphere) when referenced from published notes like [[Ghost Node Demo]].

## To Make It Visible
Add this frontmatter:
```yaml
---
publish: true
title: "Unpublished Secret Note"
tags: [demo, ghost, now-published]
---
```

Then it becomes a normal node in the graph.
